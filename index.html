e-book

- - - - - - - -
php noFrameWork mySql

Описание проекта:

Книжный магазин на PHP (большой практический localhost-проект на php). Без использования frameWork, с элементами ООП (в основном процедурное программирование). Реализация сборщика шаблона. Работа с MySQL с помощью mysqli. Самописная пагинация и валидация форм. Стороняя библиотека для captcha.
Польз. часть: вывод товаров по категориям, корзина и управление ею, оформления заказа.
Админ. часть: управление каталогом товаров, заказами, покупателями.

Технологии:

PHP, session, regex, javaScript.
MySql: sql, mysqli, подготовленные запросы.

Links:

URL демонстрационное видео на YouTube с комментариями
URL репозиторий

Подробно ▼ ▲

Пользовательская часть:

- Вывод товара:

1) Вывод товаров из БД по категориям (в соотв. с пунктом меню).
2) Вывод товаров, оформлен с учетом кол-ва блоков на странице (но не более 4-х).
3) Подсветка пунктов меню (в зависимости от типа категории).
4) Разбивка базового шаблона на подключаемые блоки.
5) Реализация пагинации страниц вывода товара (в соотв. с текущ. категорией).
6) Страница подробнее для товара.

- Корзина покупателя:

1) Вывод кол-ва заказанного товара (в панели шапки).
2) Страница обзора данных о текущем заказе:
   - Обзор корзины, удаление заказа, очистка заказа.
   - Реализация возврата на то место сайта, с которого перешли в корзину.
3) Страница оформления заказа (с вводом captcha).

Администраторская часть:

- Таблица 'books' (табл. записей товара):

1) Вывод данных (подобно PhpMyAdim),
оформлен табл. по 10-ть записей на стр. (с пагинацией).
2) Удаления товара из БД.
3) Добавления нового товара в БД и редактирование текущих данных.
4) Валидация полей при добавлении/редактировании данных товара и авторедиректы с задержкой.
5) Реализация возвратов (при обработке записей таблицы):
   - при удалении/добавлении → в конец страницы пагинации (таблицы 'books').
   - при редактировании → на ту позицию пагинации, с которой началось редактирование.
6) Подсветка пунктов меню и подменю.

- Таблица 'orders' (табл. записей заказов):

1) Оформлена таблицей по 10-ть записей на стр. (с пагинацией).
2) Возможность удалить запись.
3) Возможность посмотреть таблицу заказов, но обобщенно по покупателям.

jqReadMode

- - - - - - - -
jQuery javaScript canvas

Описание проекта:

Проект на jQuery: readMode. Пользовательский script для fireFox. Предназначенный для более удобного отображения контена. По комбинации клавиш включается режим подсветки блоков и по клику мышкой по нужному блоку с контентом, осуществляется отображения данного контента в более удобном виде (на странице отображается только данный контент, выполняется
относительный прирост шрифта).

Технологии:

jQuery, javaScript, canvas, CSS3 generated fons.
