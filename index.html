SportBlog

Описание проекта:
Типовой мини-блог реализован с помощью Yii2 (с HTML5 семантической разбивкой). Реализует вывод записей (находящихся в БД) по категориям.
Возможность посмотреть подробно информацию по записи. Вывод случайных заметок и возможность вывода новой, без перезагрузки страницы.
(проект без дизайна, css минимален)
Технологии:
HTML5, Yii2, MySQL, AJAX.
HTML5 семантическая разметка. Динамическое получение контента из БД, при помощи AJAX. Вывод "хлебных крошек" с активными URL-переходами.
Запоминание последней случайной заметки в cookie.
